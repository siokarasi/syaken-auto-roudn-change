# 車検ラウンド変更 自動監視拡張

## できること
- 一覧ページから候補行を自動抽出
- 検査日・車種・現在ラウンド単位で対象を複数選択
- 行ごとに変更先ラウンドを設定
- 約2秒前後の揺らぎ監視で自動処理
- 成功時に自動停止
- ログアウト画面検知で自動停止

## 使い方
1. Chrome の拡張機能画面を開く
2. デベロッパーモードを ON
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択
4. 対象サイトの「予約ラウンド変更」一覧ページを開く
5. 拡張Popupで「ページから再読込」を押す
6. 対象行にチェックを入れ、変更先Rを選ぶ
7. 監視間隔を設定して「監視開始」を押す

## 注意
- 実サイト操作のため、最初は対象を1件だけにして動作確認してください。
- 画面HTMLの変更により要素検出が外れる場合があります。
- フェールセーフとして連続失敗回数の上限で停止します。
